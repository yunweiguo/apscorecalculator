# Project Context: AP Psych Score Calculator

## 1. Product Positioning

This project is an SEO-first, calculator-led website targeting the keyword:

```txt
ap psych score calculator
```

The homepage is the main money page. It is not a generic brand homepage.

Current MVP positioning:

> A fast, mobile-first AP Psychology score calculator that helps students estimate their AP Psych score from 1 to 5 using MCQ and FRQ inputs.

Long-term direction:

```txt
AP Psych Score Calculator
→ AP Score Calculator Network
→ Academic Calculator Network
```

## 2. Current Strategic Focus

Do not overbuild. The current goal is not to create a full education SaaS product.

The current goal is:

```txt
Get indexed by Google
Get impressions for AP Psych score calculator keywords
Validate calculator usage
Use GSC data to decide the next pages
```

## 5. Page Roles

| URL | Page Role | Primary Intent |
|---|---|---|
| `/` | Main calculator / money page | Estimate AP Psych score |
| `/ap-psych-curve` | Curve explanation page | Understand score conversion |
| `/what-score-do-you-need-to-get-a-5` | Anxiety / reverse score page | Know target score for a 5 |
| `/how-many-questions-can-you-miss` | Anxiety / missed questions page | Estimate missed-question tolerance |
| `/ap-psych-score-distribution` | Data explanation page | Understand pass rate / 5 rate |
| `/did-i-pass-ap-psych` | Anxiety page | Estimate whether the student passed |

## 6. Product Rules

### Must keep

- Calculator above the fold on homepage
- Clear AP Psych keyword focus
- Mobile-first UI
- No login
- No database
- No CMS for MVP
- Fast loading
- Clear disclaimer
- Internal links between related pages
- FAQ sections on SEO pages
- Schema JSON-LD where relevant

### Must avoid

- Do not turn homepage into a generic academic calculator homepage yet
- Do not add AP Biology before AP Psych indexing is validated
- Do not add login, accounts, or saved history
- Do not add AI answer generation
- Do not publish many thin pages
- Do not claim official College Board affiliation

## 7. Disclaimer Requirement

Every major page should include a disclaimer similar to:

```txt
This calculator provides an estimate only and is not affiliated with or endorsed by College Board.
```

Avoid phrases such as:

```txt
Official AP Psych Score Calculator
College Board AP Psych Calculator
Guaranteed AP score
```
