# SEO Roadmap

## 1. Current Stage

The site has moved from idea to deployed MVP.

Current stage:

```txt
Stage 1: Indexing validation
```

The next goal is not feature expansion. The next goal is to validate whether Google understands and indexes the site for AP Psych score calculator queries.

## 2. 7-Day SEO Checklist

### Day 1: Indexing setup

- Bind production domain in Vercel
- Confirm homepage is accessible
- Confirm `/sitemap.xml` is accessible
- Confirm `/robots.txt` is accessible
- Add domain to Google Search Console
- Submit sitemap
- Request indexing for homepage

### Day 2: Homepage SEO check

Check that homepage has:

- Title containing `AP Psych Score Calculator 2026`
- H1: `AP Psych Score Calculator`
- Calculator above the fold
- MCQ / FRQ input fields
- FAQ section
- Internal links to support pages
- Disclaimer

### Day 3: Support page check

Check support pages:

- Each page has one primary keyword
- Each page has a direct answer near the top
- Each page links back to homepage
- Each page has FAQ
- Each page has canonical URL

### Day 4-7: GSC observation

Check:

- Indexed pages
- Impressions
- Queries
- CTR
- Average position

Do not add many new pages before GSC starts showing data.

## 3. Expansion Gates

100+ GSC impressions
First organic click received

## 8. AI Overview Defense

Every important SEO page should include:

- Direct answer
- Interactive calculator or calculator CTA
- Scenario table
- FAQ
- Internal links

Avoid pure informational pages without tools or scenarios.
